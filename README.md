# diabetes-prediction-app
Final project for Data Mining and Machine Learning, Diabetes Prediction Application. Student ID: 221001825)
About This Project
This application was developed as a final project for a Data Mining and Machine Learning course. The primary objective is to demonstrate a complete, end-to-end data science lifecycle applied to a real-world healthcare scenario.

Course Focus & Project Methodology
The pipeline applies foundational concepts of data mining and machine learning, transforming raw clinical data into an interactive, predictive web tool. Key phases of this project include:

Data Exploration & Mining: Identifying distributions, handling anomalous records, and preparing categorical features for algorithmic processing.

Feature Selection: Utilizing filter methods (ANOVA F-value) to determine the statistical significance of patient demographics and medical history.

Algorithm Training & Tuning: Comparing multiple supervised learning models (Logistic Regression, Decision Trees, Random Forest) and using Grid Search Cross-Validation to optimize hyperparameters.

Evaluation: Validating model generalization using precision, recall, and confusion matrices to minimize false negatives in a medical context.

Deployment: Serving the final tuned model through a user-friendly Streamlit web application.
